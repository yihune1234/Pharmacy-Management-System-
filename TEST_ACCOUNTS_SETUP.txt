Issue: "Accounts already exist"
  → Accounts were already created
  → You can still login with the credentials
  → Or delete accounts from database and recreate

Issue: "Role not found"
  → Roles table might be empty
  → Run database migrations first
  → Check roles table has admin, pharmacist, cashier

Issue: "Database error"
  → Check database connection
  → Verify employee table exists
  → Check database permissions

Issue: Login not working
  → Verify credentials are correct
  → Check database connection
  → Verify employee table has the accounts
